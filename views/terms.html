Termeni si conditii de utilizare

Termenii si Conditiile generale de utilizare stabilesc termenii si conditiile generale de utilizare a site-ului „redirectioneaza.ro” de catre potentialii vizitatori sau beneficiari.

“Redirectioneaza.ro” (in continuare denumit „Site”, „Platforma”, „Redirectioneaza.ro”) este o platforma online dezvoltata de Asociatia Code for Romania (denumita in continuare “Code for Romania”) care vine in sprijinul actiunilor civice de redirectionare a pana la 3.5% din impozitul anual pe veniturile din salarii si pensii, avand ca obiectiv facilitarea relatiei dintre asociatiile non-guvernamentale sau fundatiile (in continuare denumite „ONG”) infiintate in Romania pe de o parte si potentialele persoane care doresc sa redirectioneze pana la 3.% catre acestea, pe de alta parte.

Asociatia Code for Romania este o persoana juridica, avand sediul in Mun. Bucuresti, Piata Alba Iulia nr. 7, bloc I6, etaj 1, ap. 6, inregistrata in registrul special, Partea A, sectiunea I 67/06.07.2016.

Obiectivul Site-ului

Obiectivul Platformei este de a facilita colectarea formularelor de tip 230, formulare ce permit redirectionarea sumei reprezentand pana la 3.5% din impozitul anual pe venit, precum si incurajarea cetatenilor sa contribuie la colectarea acestora.

  • Site-ul permite inregistrarea unui ONG pentru a crea un profil online prin care acesta se poate prezenta.
  • De asemenea, Redirectioneaza.ro pune la dispozitia utilizatorului Platformei care doreste sa redirectioneze pana la 3.5% din impozitul pe venitul, un formular care colecteaza datele personale ale acestuia. Pentru a afla conditiile in care datele cu caracter personal ale utilizatorilor platformei care doresc sa directioneze pana la 3.5% din impozitul pe venit, sunt utilizate de catre Code for Romania, te rugam sa accesezi Nota de informare de aici.
  • Datele din acest formular sunt folosite pentru a completa documentul de tip 230, cu datele ONG-ului, dar si datele persoanei care Redirectioneaza cei pana la 3.5%.
  • Astfel, odata completat formularul, platforma genereaza formularul tip 230 cu datele completate de tine, putand fi descarcat de pe Site automat (vezi exemplu necompletat).

In cazul in care utilizatorul nu doreste ca formularul tip 230 sa fie precompletat in baza formularului de pe Platforma, sau sa ii fie prelucrate datele cu caracter personal, poate alege sa redirectioneze cei pana la 3.5% din impozit descarcand si completand formularul tip 230 de pe site (vezi exemplu necompletat).

Despre Code for Romania

Code for Romania este o organizatie neguvernamentala independenta, neafiliata politic si apolitica, infiintata in anul 2016, al carei scop este de a mobiliza tinerii profesionisti din domeniul IT sa creeze instrumente digitale care ajuta in solutionarea problemelor sociale. Code for Romania este infiintata si functioneaza in conformitate cu legea romana, actul constitutiv si statutul sau si codul de conduita.
